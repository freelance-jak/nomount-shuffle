# ノーマウント登壇者シャッフルアプリ(フロント)

## 本番 URL

[https://nomount-shuffle.vercel.app/](https://nomount-shuffle.vercel.app/)

## セットアップ方法

```
git clone https://github.com/reachscript-jak/nomount-shuffle
cd nomount-shuffle
yarn
yarn dev
```

・上記実行後、開発環境にアクセス
http://localhost:3000

## 利用技術

- Next.js
- TypeScript
- ESLint
- Prettier
- Jest
- Tailwind CSS
